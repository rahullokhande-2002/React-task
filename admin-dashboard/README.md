Admin dashboard :
this is simple admin dashboard build by using react . The project  focuses  on implemented the authentication, user management, and a clean folder structure

Tech Stack Used :
React JS
React Router DOM
Context API
JavaScript (ES6)
Bootstrap (for styling)
LocalStorage (for data persistence)

Setup Instructions :
Clone the repository
git clone https://github.com/your-username/admin-dashboard.git ,
Navigate to project folder ,
cd admin-dashboard,
Install dependencies,
npm install,
Run the project,
npm run dev
